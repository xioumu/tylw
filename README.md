#一个论文审查系统
=====
##使用工具
=========
xampp v1.8.2<br/>
php v5.4.19<br/>
bootstrap 2<br/>
Javascript<br/>
JQuery<br>
charisma https://github.com/xioumu/charisma <br/>
##文件大概结构
=========
./
    css/    放CSS文件
    doc/    放一些样例表格
    img/    放要用的图标
    js/     放JS文件
    phpExcel/   phpExcel类，用来存取EXCEL
    upFile/     存放上传的文件
        exl/    上传的EXL表格
        paper/  上传的论文
        report/ 上传的开题报告
    verification/   验证码类
    addEva.php  添加审评
